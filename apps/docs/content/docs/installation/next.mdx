---
title: next.js
description: Install and configure duck/ui for Next.js.
---

<Steps>

<Step>Create project</Step>

Run the `init` command to create a new Next.js project or to setup an existing one:

```bash
npx @gentleduck/cli init
```

Choose between a Next.js project or a Monorepo.

<Step>Add Components</Step>

You can now start adding components to your project.

```bash
npx @gentleduck/cli add button
```

The command above will add the `Button` component to your project. You can then import it like this:

```tsx {1,6} showLineNumbers title="app/page.tsx"
import { Button } from "@/components/ui/button"

export default function Home() {
  return (
    <div>
      <Button>Click me</Button>
    </div>
  )
}
```

</Steps>
