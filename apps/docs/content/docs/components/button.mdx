## Features

- Multiple **styles**, **sizes**, and **border** options out of the box
- Built-in **loading state** with spinner and auto-disable
- Supports **icons**, **dual icons**, and **collapsed icon-only mode**
- Flexible `asChild` rendering for links and custom wrappers
- Smooth **hover animations** with optional `AnimationIcon`
- Fully **typed**, **accessible**, and **responsive**
- Powered by `@gentleduck/variants` for scalable theming
- Clean, composable, and **Tailwind-optimized**

```tsx
import { Button } from '@/components/ui/button'
```


### Button

`Button` is a versatile and customizable React component designed to render a button with various styles and functionalities. It supports different sizes, variants, and additional features like loading states and icons. The component is flexible and can be used in a variety of scenarios, from simple buttons to complex interactive elements.

#### Example Usage:

```tsx {5, 10}
import { Button } from '@/components/ui/button'

const MyButton = () => {
  return (
    <Tooltip>
      <Button size="large" variant="primary">
        Submit
      </Button>
      <TooltipContent>Submit Button</TooltipContent>
    </Tooltip>
  )
}
```

### Link

You can use the `buttonVariants` helper to create a any element like a link that looks like a button.

```tsx
import { buttonVariants } from '@/components/ui'
```

```tsx
<Link className={buttonVariants({ variant: 'outline' })}>I'am a Link ðŸ¦†</Link>
```

Alternatively, you can set the `asChild` parameter and nest the link component.

```tsx

  <Link href="/tos">I'am a Link with tos ðŸ¦†</Link>
</Button>
```

## API Reference

### General Button Properties

* **`variant`** (`'default' | 'destructive' | 'warning' | 'outline' | 'dashed' | 'secondary' | 'ghost' | 'link' | 'expand_icon' | 'nothing'`, optional):

  * Defines the button style.
  * `'default'`: Solid primary button.
  * `'destructive'`: Red destructive action.
  * `'warning'`: Yellow warning action.
  * `'outline'`: Outlined with border.
  * `'dashed'`: Dashed outline variant.
  * `'secondary'`: Muted secondary style.
  * `'ghost'`: Transparent with hover effect.
  * `'link'`: Looks like a hyperlink.
  * `'expand_icon'`: Icon-expanding group button.
  * `'nothing'`: Unstyled button (useful for resets).

* **`size`** (`'icon' | 'xs' | 'sm' | 'default' | 'lg' | 'xl' | '2xl' | '3xl'`, optional):

  * Defines the button size.
  * `'icon'`: Icon-only square button.
  * `'xs'` â†’ `'3xl'`: From extra small to extra large sizes.

* **`border`** (`'default' | 'primary' | 'secondary' | 'destructive' | 'warning'`, optional):

  * Defines the border and background combination.
  * `'default'`: No additional border styles.
  * `'primary'`: Subtle primary border.
  * `'secondary'`: Secondary soft background.
  * `'destructive'`: Semi-transparent red border.
  * `'warning'`: Semi-transparent yellow border.

* **`asChild`** (`boolean`, optional):

  * If `true`, renders using a custom child element via `Slot` (e.g. a `Link`).

* **`loading`** (`boolean`, optional):

  * If `true`, shows a spinning loader icon and disables the button.

* **`disabled`** (`boolean`, optional):

  * Disables the button manually.

* **`icon`** (`React.ReactNode`, optional):

  * Icon placed before the content (left-aligned or centered if collapsed).

* **`secondIcon`** (`React.ReactNode`, optional):

  * Icon placed after the content (right-aligned).

* **`isCollapsed`** (`boolean`, optional):

  * If `true`, renders icon-only button (hides children and secondIcon).

* **`type`** (`'button' | 'submit' | 'reset'`, optional):

  * Native HTML button type. Default is `'button'`.

* **`ref`** (`React.Ref<HTMLButtonElement>`, optional):

  * Ref forwarding support for imperative access.

* **`className`** (`string`, optional):

  * Custom class name for styling overrides.

* **`...props`** (`React.HTMLProps<HTMLButtonElement>`):

  * All native HTML button props are supported.

---

### AnimationIcon Properties

* **`children`** (`React.ReactNode`, required):

  * The main content or label inside the component.

* **`animationIcon`** (`{ icon?: React.ReactNode; iconPlacement?: 'left' | 'right' }`, optional):

  * Defines an animated icon on hover.
  * `icon`: The icon to animate.
  * `iconPlacement`: `'left'` or `'right'`, determines icon direction and placement.
