## API Reference

### **ScrollArea**

* **`viewportClassName`** (`string`, optional): Additional CSS classes to apply to the scroll viewport container.
* **`...props`** (`React.HTMLAttributes<HTMLDivElement>`): Other native props forwarded to the outer container `<div>`.
