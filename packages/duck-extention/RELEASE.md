### Changelog

**Major Features:**
* Per-domain font selection: Each website can have its own custom font
* Floating toggle button: Added a floating button on every website to quickly enable/disable the extension
* Domain-specific management: View and manage fonts for all configured websites in one place
* Real-time synchronization: Changes apply instantly across all tabs without page reload

**Features:**
* Toggle enable/disable per domain directly from the popup
* Visual indicators for current domain and disabled domains
* Corner border highlight for the current domain
* Improved domain detection and validation
* Better UI organization with domain list management

**Improvements:**
* Cleaner codebase
* More intuitive user interface
* Better state management with per-domain storage
* Enhanced domain matching logic

**Removed:**
* Global font setting (replaced with per-domain)
* Copy CSS button
