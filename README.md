## Getting Started

Clone the repo:

```bash
git clone https://github.com/gentleeduck/duck-ui.git
cd duck-ui
```

Install dependencies (using [pnpm](https://pnpm.io) recommended):

```bash
pnpm install
```

Build all packages:

```bash
pnpm build
```

Run the local dev environment:

```bash
pnpm dev
```
